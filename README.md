# Overview 

## Mini Shell (GTK4 + C)
Overview

Mini Shell is a simple graphical Linux shell built using C, GTK4, and standard Unix system calls. It provides a terminal-like interface inside a GTK window where users can execute Linux commands and view their output.

This project was created to learn:

- Process creation with fork()
- Program execution with execvp()
- Inter-process communication using pipe()
- File descriptor redirection using dup2()
- Process synchronization using waitpid()
- GTK4 GUI programming

# Learning Objectives

This project helped me understand:

- Linux process management
- Parent-child processes
- Pipes and IPC
- File descriptors
- Output redirection
- GTK4 event handling
- Building graphical applications in C
